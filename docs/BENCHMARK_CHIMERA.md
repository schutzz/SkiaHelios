## üìä Module Performance Insights

### 1. AION (Persistence)
* **Performance:** Excellent.
* **Observation:** Successfully parsed `Autoruns.csv` despite non-standard headers and encoding issues. Identified the malicious WMI consumer immediately.

### 2. Chronos (Time Verification)
* **Performance:** **Hyper-Sensitive (High Noise Floor)**
* **Detection Count:** 40,445 anomalies flagged.
* **Analysis:** * **Success:** Successfully detected the simulated timestomping attack (T1070.006) on the target payload.
    * **Challenge:** The module currently flags *any* nanosecond discrepancy between `$STANDARD_INFORMATION` and `$FILE_NAME`. In a Windows environment, legitimate background processes (TrustedInstaller, Windows Update, Log rotation) frequently create these discrepancies, resulting in a high False Positive rate.
* **Verdict:** While effective at detecting the attack, the current logic requires significant manual filtering.
* **Roadmap (v3.0):** Implementation of "Smart Filtering" to prioritize **Executables (.exe, .dll, .sys)** and **Scripts (.ps1, .bat)**, ignoring low-risk file types (logs, caches) to improve Signal-to-Noise ratio.

### 3. Sphinx (Deciphering)
* **Performance:** Accurate.
* **Observation:** Successfully extracted and decoded the Base64 payload from PowerShell Event Logs (Event ID 4104). The entropy threshold adjustment (v1.2) was critical for catching shorter obfuscated strings.

### 4. Plutos (Exfiltration)
* **Performance:** Precise.
* **Observation:** Correctly identified `OneDrive.exe` activity and the target script `set-persistenceaudit.ps1` as high-risk exfiltration candidates based on directory context.
